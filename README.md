# UTG Manufacturing Analytics Dashboard

## Project Overview
This project is a Power BI-based Manufacturing Analytics Dashboard developed for United Toughened Glass (UTG) using mock operational data.

The dashboard provides real-time business insights into:
- Revenue performance
- Production workflow
- Order tracking
- Dispatch monitoring
- Operational delays
- Factory-wise analysis

The goal of this project is to demonstrate how data analytics and business intelligence can improve manufacturing visibility and operational decision-making.

---

## Tools & Technologies Used
- Power BI
- Excel
- SQL Concepts
- Data Visualization
- Business Intelligence

---

## Key Features
- Interactive KPI Cards
- Factory-wise Revenue Analysis
- Production Workflow Funnel
- Delayed Order Monitoring
- Dispatch Tracking
- Product & Region Filters
- Operational Insights Dashboard

---

## Dashboard KPIs
- Total Revenue
- Total Orders
- Delayed Orders
- Average Wastage
- Delivered Orders
- Dispatch Pending

---

## Business Insights
- Punjab Unit generated the highest revenue
- Operational bottlenecks identified in dispatch workflow
- Delay trends monitored across production stages
- Factory-wise comparison enabled performance evaluation
