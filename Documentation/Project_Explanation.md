# Project Explanation

## Objective
The objective of this dashboard is to provide operational visibility into manufacturing workflows and business performance.

## Problems Addressed
- Lack of production monitoring
- Difficulty tracking delayed orders
- Limited factory-wise performance visibility
- Inefficient dispatch monitoring

## Solution
An interactive Power BI dashboard was developed to monitor:
- Revenue
- Orders
- Production stages
- Dispatch status
- Delays
- Factory performance

## Outcome
The dashboard enables management-level visibility and supports operational decision-making through centralized reporting.
